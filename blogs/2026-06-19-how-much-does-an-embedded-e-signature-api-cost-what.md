This guide examines how embedded e-signature API pricing affects ISV margins, since every envelope or signature request can cut into product unit economics. It outlines common billing units (envelopes, documents, signature requests, seats) and presents an ISV margin equation: customer price per envelope minus API cost minus operational expenses equals retained margin. It stresses that volume behavior and partnership terms (white-label rights, overage structures, declining per-unit costs at scale) often matter more than starter pricing.
